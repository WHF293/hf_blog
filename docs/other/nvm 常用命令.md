## NVM

NVM 是一个 node 环境管理工具

NVM 基本命令

```shell
// 查看当前使用的node版本
nvm current 
// 查看所有本地可用的node版本
nvm ls 
// 切换 node 版本
nvm use xx.xx.xx 
// 下载指定版本的 node
nvm install xx.xx.xx
// 卸载指定版本的 node
nvm unInstall xx.xx.xx
// 在当前node版本中，将npm升级到最新版
nvm install-latest-npm 
```
